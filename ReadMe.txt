Folder zawiera główny kod programu oraz biblioteki potrzebne do uruchomienia tego programu.
Zawartość tego folderu jest częścią pracy inżynierskiej wykonywanej pod kierownictwem dr. inż. Mirosława Miciaka.

main.py			-	główny kod programu
lib\ds3231_i2c.py	-	biblioteka do obsługi modułu zegara czasu rzeczywistego
lib\mfrc522.py		-	biblioteka do obsługi modułu identyfikacji radiowej
lib\umqtt\simple.py	-	biblioteka do możłiwości połączenia z brokerem MQTT
lib\stdin\_init_.py	-	plik systemowy Raspberry Pi Pico W
